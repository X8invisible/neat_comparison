Prototype Application for the "Comparison of two evolutionary approaches
in training a Neural Network to play a game" Honours Project

The application requires Python3 installed and a working terminal with access to python and pip.
If on a non-Windows machine the commands listed below might be required to be changed to "python3" or "pip3" accordingly.



To run the program:
1. Make sure all the dependencies are installed by running the command:
pip install -r requirements.txt

NOTE. in order to visualise the graphs, graphviz will have to be installed separately on your system and configured on the PATH variable (for windows)
Graphviz can be downloaded here: https://www.graphviz.org/download/

2. To start the application run the following command:
python app.py
